1. 总体流程总览

图 1. 从数据获取到背景孔径流量产出的全链路。

2. 区域下载模块

脚本 download_d3_region_formal.py 通过 TAP 服务在圆形天区内查询指定波段的观测，并并行下载 FITS。

核心空间条件：

$$\mathrm{INTERSECTS}(p.poly,\ \mathrm{CIRCLE}(\mathrm{ICRS},\alpha_0,\delta_0,r))=1$$

角距离形式等价写法：

$$\cos(\Delta\theta)=\sin\delta\sin\delta_0+\cos\delta\cos\delta_0\cos(\alpha-\alpha_0),\ \Delta\theta\le r$$

同时按 obs_id 去重，避免重复下载。

3. 3x3中心像素WCS模块

脚本 pixel_to_wcs_3x3_centers.py 选取每个 3x3 block 的中心像素并转换为 WCS 坐标。

采样规则（0-based）：

$$x_k=1+3k,\quad y_m=1+3m$$

block 索引回写：

$$\mathrm{block\_ix}=\left\lfloor\frac{x-1}{3}\right\rfloor,\quad \mathrm{block\_iy}=\left\lfloor\frac{y-1}{3}\right\rfloor$$

坐标映射：

$$(\mathrm{ra\_deg},\mathrm{dec\_deg})=\mathrm{WCS}_{celestial}(x,y)$$

4. SEP源掩膜模块

脚本 replace_hdu_sep_mask.py 对 D2/D3 图像执行 SEP 探测并掩膜，源像元设为 NaN。

检测与掩膜逻辑：

  • 先用 sep.Background 获取背景模型
  • 执行 sep.extract(..., segmentation_map=True)
  • 使用 segmap > 0 的区域作为源掩膜

阈值参数由 sigma 控制，常用 3。

5. 背景孔径流量提取模块

脚本 extract_d3_spectra_spxquery.py 在掩膜后图像上提取背景孔径流量，并同时输出两种误差。

5.1 孔径流量

$$\mathrm{aperture\_sum\_flux\_ujy}=\left(\sum_{i\in ap} I_i\right)A_{pix}$$

5.2 纯方差传播误差

$$\mathrm{aperture\_sum\_flux\_error\_var\_ujy}=\sqrt{\sum_{i\in ap}\mathrm{Var}_i}\,A_{pix}$$

5.3 方差+背景波动二次和误差

$$s_{ap}=\mathrm{std}(I_{ap}),\quad \sigma_{bg}=N_{ap}\,s_{ap}$$

$$\mathrm{aperture\_sum\_flux\_error\_var\_bgquad\_ujy}=\sqrt{\left(\sqrt{\sum\mathrm{Var}_i}\right)^2+\sigma_{bg}^2}\,A_{pix}$$

5.4 背景孔径SNR

$$\mathrm{aperture\_sum\_snr}=\frac{\mathrm{aperture\_sum\_flux\_ujy}}{\mathrm{aperture\_sum\_flux\_error\_var\_bgquad\_ujy}}$$

兼容列 aperture_sum_flux_error_ujy 当前等于 aperture_sum_flux_error_var_bgquad_ujy。

6. 核心公式汇总

模块变量/关系公式
空间查询 圆形天区选择 INTERSECTS(poly, CIRCLE(ICRS, alpha0, delta0, r)) = 1
3x3采样 中心像素 x = 1 + 3k, y = 1 + 3m
WCS转换 像素到天球 (ra, dec) = WCS_celestial(x, y)
背景流量 孔径求和 aperture_sum_flux_ujy = (sum I_i) * A_pix
误差A 纯方差传播 sqrt(sum Var_i) * A_pix
误差B 方差+背景波动二次和 sqrt((sqrt(sum Var_i))^2 + (N_ap * std(I_ap))^2) * A_pix

7. 推荐命令模板

7.1 下载 D2

conda run -n spxquery python /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/code/download_d3_region_formal.py \
  --center-ra 32.5 --center-dec 0.9 --radius-deg 1.5 \
  --band D2 --output-dir /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/data/formal/D2 \
  --max-workers 8

7.2 下载 D3

conda run -n spxquery python /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/code/download_d3_region_formal.py \
  --center-ra 32.5 --center-dec 0.9 --radius-deg 1.5 \
  --band D3 --output-dir /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/data/formal/D3 \
  --max-workers 8

7.3 生成3x3中心像素WCS CSV

conda run -n spxquery python /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/code/pixel_to_wcs_3x3_centers.py \
  --input-fits /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/data/formal/D3/2025W30_1B_0184_1.fits \
  --output-csv /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/code/2025W30_1B_0184_1_pixel3_center_wcs.csv

7.4 SEP掩膜 D2/D3

conda run -n spxquery python /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/code/replace_hdu_sep_mask.py \
  --input-dir /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/data/formal/D2 \
  --output-dir /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/output/replaced_image/D2 \
  --sigma 3

conda run -n spxquery python /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/code/replace_hdu_sep_mask.py \
  --input-dir /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/data/formal/D3 \
  --output-dir /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/output/replaced_image/D3 \
  --sigma 3

7.5 背景流量提取（8核）

conda run -n spxquery python /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/code/extract_d3_spectra_spxquery.py \
  --fits-dirs /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/output/replaced_image/D2 \
              /data1/longlin/workspace/spherex_halpha_overdensity/ha_cube/output/replaced_image/D3 \
  --allowed-detectors 2,3 \
  --output /data1/longlin/workspace/spherex_halpha_overdensity/OIII_cube/result/spectra_all_D2D3.csv \
  --output-dir /data1/longlin/workspace/spherex_halpha_overdensity/OIII_cube/result/spectra \
  --workers 8 --aperture-method fixed --aperture-diameter 3 \
  --background-method annulus --min-bg-pixels 30
